Python中的scalar与Non-scalar¶

scalar类型数据，其特征是唯一，不可再分，如数值5.¶

Non-scalar类型数据，其特征是可以继续划分，如数组{1,2,3,4}可以划分为其他的子数组，如{1},{2}等等。¶

求解距离矩阵¶

In [15]:
# 求解距离矩阵
n = 100   # 样本个数
x = np.linspace(1, 100, n)    # 样本的横坐标   linspace()是生成等差数组的函数 linspace=linear space，译为线性等分向量
y = np.linspace(1, 100, n)    # 样本的纵坐标

dist = np.zeros([n, n])       # 初始距离矩阵

for i in range(n):
    for j in range(n):
        dist[i, j] = np.sqrt((x[i] - x[j])**2 + (y[i] - y[j])**2)   # 计算欧式距离

print(x)
print(y)
print(dist)

数组的形态变化¶

1.形态变化前后的数组元素个数必须相等
2.flatten()函数是一个矩阵降维的函数
3.必须注意flatten()函数的参数和使用方法

In [27]:

arr4 = np.arange(1, 13) #生成12个元素
print(arr4)
print(arr4.reshape([3, 4]))  # 改变数组的shape 3行4列的数组
print(arr4.reshape([4, 3])) #4行3列
arr5 = arr4.reshape([3, 4])   # 生成二维数组
print(arr5)
print(arr5.ravel())   # 数组展平
arr5.flatten()
arr6 = arr4.reshape([3, 4])
print(arr6)
print(np.vstack([arr5, arr6]))    # 纵向组合（拼接）
print(np.hstack((arr5, arr6)))    # 横向组合（拼接）
